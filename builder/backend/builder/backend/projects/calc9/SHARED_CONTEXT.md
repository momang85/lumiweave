# 计算器项目接口约定

## 文件结构
- `projects/calc9/frontend/index.html` - 主页面
- `projects/calc9/frontend/style.css` - 样式文件
- `projects/calc9/frontend/app.js` - 逻辑文件

## 功能说明
- 支持加减乘除四则运算
- 支持键盘输入（数字、运算符、Enter=、Backspace 删除、Esc 清除）
- 支持点击按钮操作
- 显示当前输入和上一步运算

## 数据格式
- 当前显示：字符串格式数字
- 运算结果：字符串格式数字
- 错误处理：除以零时弹出提示并清空

## 使用方式
直接在浏览器打开 index.html 即可使用
