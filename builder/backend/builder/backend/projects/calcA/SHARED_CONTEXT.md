# 计算器项目接口约定

## 文件结构
- `frontend/index.html` - 主页面
- `frontend/style.css` - 样式文件
- `frontend/app.js` - JavaScript逻辑

## 功能说明
- 支持四则运算：加(+)、减(-)、乘(×)、除(÷)
- 支持AC(清除)、DEL(删除)功能
- 支持键盘输入和鼠标点击
- 显示当前输入和之前的运算

## 数据格式
- 当前操作数：字符串格式，支持小数
- 之前操作数：字符串格式
- 运算符：+、-、×、÷

## 事件处理
- 按钮点击事件
- 键盘事件支持
- 错误处理：除以零时弹出提示
