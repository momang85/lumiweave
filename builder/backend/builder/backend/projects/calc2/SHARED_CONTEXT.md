# 计算器接口约定

## 显示屏字段
- `display`: string类型，显示当前输入或计算结果
- 最大显示15个字符，超出部分截断

## 按钮操作
- 数字按钮 (0-9): 追加数字到显示屏
- 运算符按钮 (+, -, *, /): 记录操作符和第一个操作数
- 清除按钮 (C): 清空显示屏
- 等于按钮 (=): 执行计算并显示结果

## 计算逻辑
- 支持连续计算：5 + 3 + 2 = 10
- 除法时检查除数不为0
- 结果保留最多6位小数

## 样式要求
- 显示屏：黑色背景，白色文字，右对齐
- 按钮：圆角矩形，hover效果，active状态变化
- 布局：响应式，移动端友好
